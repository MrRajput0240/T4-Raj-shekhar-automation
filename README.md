# T4-Raj-shekhar-automation
company automation assessment 
* **These are following System  software is required For the project**

1. Eclipse IDE
2. Add JDK file as its Java selenium (>= JDK 8 )
3.import Selenium WebDriver
4.launch Chrome Browser
5. Good internet connection
6. Steps We Performed in the Project:

* **Login Module**
1. First I open the Chrome browser.
2. Then I enter the application URL.
3. After that I give valid email id and password.
4. Then I click on Login button.
5. If login is successful,  then we will open  the dashboard page.
6. I confirm that login is working properly.

* **Add New Generator**
1. After login, I go to Generator Management section.
2. Then I click on Add New Generator button.
3. Then I enter Generator Name.
4. Then I enter Internal Account Number.
5. After filling details, I click on Save button.
6. I check whether generator is created successfully or not.

* **Billing Information**
Then I go to Billing Information section.
I select Industry from dropdown.
I enter the valid Street Address.
I give the valid Email id.
I enter the valid Phone number.
Then I save the details.
I check all values are entered correctly.

* **Latitude and Longitude**
Then I click on Generate button for latitude and longitude.
System automatically fills both fields.
I verify that both the latitude and longitude fields are not empty.
If values are coming , that means working fine.

* **Copy Billing Information**
Then I click on Copy Billing Information option.
After clicking, data should copy automatically.
I verify copied details are matching.

* **Service Hours**
Then I enter Open time.
I enter Lunch time.
I enter Close time.
Then I click on Copy to All Days option.
I check whether all days timing updated properly.

* **Route Assignment**
Then I go to Route Assignment section.
I click on Add Service.
I select Route name.
I select Service Type as Medical Waste.
I select Frequency as Multiple Times Weekly.
I select weekdays (Thu, Fri, Sat, Sun).
I select 2 containers in Scope of Work.
Then I save the service.
I verify service is added successfully.

* **Price Book**
Then I go to Price Book section.
I enter service price.
I update the price.
Then I save the changes.
I verify price updated correctly.

* **Logout**
Finally I click on profile icon.
Then I click on Logout.
Application should redirect to login page.
I confirm logout is successful.
