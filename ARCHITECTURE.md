**About The Project :**

This project covers complete end-to-end automation of the application workflow. It includes generator creation, billing setup, service configuration, route allocation, price book modification, and logout process. The automation ensures all mandatory fields are validated and confirms that the entire workflow runs successfully through browser automation.
**Technology Used in Project**
**Selenium WebDriver 4** – Used to automate web browser actions like clicking buttons, entering input data, navigating between pages, handling dropdowns, and verifying application functionality.
Java – Used for developing automation scripts and implementing the test logic for execution and validation.
**IDE:**
Eclipse IDE – Used for writing, editing, debugging, and running the automation test scripts.
**Browser:**
Latest version of Google Chrome – Used to execute and validate automated test scenarios.
**Driver:**
Chrome Driver – Acts as a bridge between Selenium WebDriver and the Chrome browser to perform automation actions.
**Version Control:**
Git – Used to maintain source code, manage versions, and track changes in the automation framework.
